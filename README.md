# 🎲 Backgammon Royal

**"A passion project in the making – bringing the classic game of Backgammon to the digital age."**

### 🚧 Status: Early Development
Welcome to the early stages of **Backgammon Royal**! I have just begun this journey to build a robust, professional, and engaging Backgammon platform. While the board is currently being drawn, the vision is set. 

### 🎯 My Goals
As a Software Engineering student, my mission here is to translate theoretical concepts into a complete, high-quality full-stack system:
* **Custom Game Engine:** Implementing a secure, fair, and rule-compliant Backgammon engine from scratch.
* **Real-time Multiplayer:** Connecting players using WebSocket technology.
* **Intelligent AI:** Developing a bot that strategically challenges players.

### 🛠 The Tech Stack
* **Frontend:** React Native (Expo)
* **Backend:** Node.js & Socket.io 
* **Logic:** Custom TypeScript engine

### 🌟 Why Backgammon?
I've been a Backgammon enthusiast for years. I find the balance between probability (the dice) and strategy (the movement) fascinating. Building this is my way of diving deep into systems programming and game logic outside of my regular university projects at BGU.
